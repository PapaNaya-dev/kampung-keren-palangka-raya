## Format Penamaan File
File logo harus mengikuti format berikut:
```
LOGO_NamaUniversitas_NamaKawasan.png
```

## Catatan:
- Gunakan format PNG untuk transparansi background
- Ukuran logo disarankan maksimal 200x200 pixel
- Logo akan ditampilkan dengan ukuran 80x80 pixel di popup
- Jika logo tidak ditemukan, akan ditampilkan placeholder (buat file placeholder jika diperlukan)
- Nama universitas dapat disesuaikan dengan data sebenarnya di mapping JavaScript pada index.html
